# Open Ticketer

An open-source alternative to services like Ticket Master.

## Features

- Sell tickets online for events with a link
    - Not included yet: Picking a seat
    - Purchasing goes through Stripe
    - Tickets are sent over email and a single QR code for a group
      - Can contain multiple entries
- Scan tickets with any smartphone device
    - Responds with:
        - who the ticket is for
    - Check off the individuals as they're admitted
- Transfer your tickets
  - Accounts with a high number of transfers are flagged
- Organization management
    - Manage users (including who can scan tickets)
    - Manage events
